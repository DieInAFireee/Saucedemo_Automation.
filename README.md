Running Tests
To run a specific test, open Git Bash in the project directory and execute the following command:
npx wdio run wdio.conf.js --spec ./test/specs/sauceDemoLoginX.js
Replace X with the number of the test you want to run.
Run test case 1:
npx wdio run wdio.conf.js --spec ./test/specs/sauceDemoLogin1.js
